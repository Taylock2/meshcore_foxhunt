# Fox hunt
A basic html page that helps read raw packets from ble meshcore companions.
Filter to 0 hops and a specific channel hash to log the gps position + rssi of received signal.
An attempt to locate the source of the messages will be completed on the map.

If you are trying to perform this on an encrypted channel you will not be able to tell who the sender is, so if more then one are in range it will cause issues with the tracking.
Also try to keep on open field of view so as to not obstruct the signal.

# Happy hunting
